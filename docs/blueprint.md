# **App Name**: FactuLab

## Core Features:

- Formulario de Inventario Clínico: Entrada de datos detallada para nombre, marca, modelo y número de serie del equipo de laboratorio.
- Generador de PDF del Cliente: Creación instantánea de facturas PDF en el navegador utilizando recursos locales sin dependencias de servidor.
- Validador de Integridad de Datos: Validación automática de campos para prevenir campos vacíos antes de la exportación del documento.
- Gestor de Marca y Folio: Sistema para asignar números de factura únicos e incrustar automáticamente el logotipo tienda.png en el encabezado.
- Exportación Directa: Botón nativo para compartir o guardar localmente el documento PDF generado.
- Asistente de Atributos Sugeridos: Un 'tool' inteligente que sugiere especificaciones técnicas relevantes basadas en el modelo de equipo ingresado.

## Style Guidelines:

- Esquema de color claro con un azul técnico primario (#1F3354) para transmitir profesionalismo y precisión.
- Fondo suave y desaturado (#EFF1F4) para minimizar la fatiga visual en entornos de laboratorio.
- Color de acento en azul cian vibrante (#29D6FF) para resaltar botones de acción y validaciones exitosas.
- Se utiliza 'Inter' como fuente principal por su aspecto moderno, mecanizado y objetivo, ideal para contextos científicos y técnicos.
- Iconos minimalistas de línea delgada representativos de equipos de laboratorio y herramientas de exportación.
- Diseño de formulario de columna única centrada para facilitar la entrada de datos en dispositivos móviles y web.
- Transiciones suaves de opacidad al generar la previsualización del PDF para confirmar la creación del archivo.
